setWidgetView()method 
public function setWidgetView(widget:*):void

This is used both to add new widgets to a page, alter the condition of existing ones and to remove unwanted widgets. If a widget with the provided name does not exist, it will be created. If it already exists, and the provided view is different than the existing one, it will be tweened to that view. If a widget already exits (from a previous page), but is not specified, it will be tweened to a special "destroy" view on the end of which it'll be removed.
