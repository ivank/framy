newPoly()method 
public static function newPoly(points:Array, options:Object = null, attributes:Object = null):fySprite

Creates a new fySprite and draws a polygon inside it - the polygon is represented by an array of endpoints (corners) Using this you can draw triangles, rectangles, pentagons, stars, etc ...

Parameters
points:Array — An array of points (using the Point class)
 
options:Object (default = null) — A hash of options, used to fill the polygon, accepts color and alpha
 
attributes:Object (default = null) — Applies thoes attributes to the whole fySprite after drawing the rectangle

Returns
fySprite — The created fySprite

See also

newRect
newFrame
newCircle
flash.geom.Point

Example
Create an orange trinagle:
import flash.geom.Point

fySprite.newPoly([new Point(0,0), new Point(100,50), new Point(0,100)], { color: 'orange' })
